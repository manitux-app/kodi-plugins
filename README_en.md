#kodi-plugins
Turkish content provider plugins for Kodi

## Kodi Repository

Repository add-on zip:

https://raw.githubusercontent.com/manitux-app/kodi-plugins/main/repository.manitux-1.0.0.zip

https://tinyurl.com/mntxrepo

After installing this zip file in Kodi using `Install from zip file`, plugins can be installed from the `Manitux Repository`.

## Short Code Installer

Installer plugin zip:

https://raw.githubusercontent.com/manitux-app/kodi-plugins/main/script.manitux.repoinstaller-1.0.2.zip

When this plugin is opened, select `Add Repository` or `Add Plugin`. Only short codes like `mntxrepo` or the full short URL can be entered into the dialog window. The extension extracts the short address to the long zip address, downloads it, and installs only the repository or plugin to Kodi according to the selected type.
